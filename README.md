## Projetos em destaque

### Timely Calendar

Aplicação em Angular para visualização de eventos em um calendário mensal interativo, com consumo de API, filtros por data, categorias e tags, modal de detalhes, tratamento de loading/erro e testes unitários.

**Tecnologias:** Angular, TypeScript, Angular Material, RxJS, SCSS, Jasmine e Karma

[Repositório](https://github.com/CardosoRepository/Timely-Calendar) | [Deploy](https://cardosorepository.github.io/Timely-Calendar/)

### Kogui Challenge

Aplicação em Angular baseada na API de Rick and Morty, com autenticação local, rotas protegidas, busca global, perfil de usuário, listagem de personagens, episódios e locais.

**Tecnologias:** Angular, TypeScript, RxJS, SCSS, Bootstrap Icons e Rick and Morty API

[Repositório](https://github.com/CardosoRepository/kogui-challenge) | [Deploy](https://kogui-challenge.netlify.app)

### Letmeask

Aplicação em React para criação de salas de perguntas e respostas em tempo real, com autenticação via Google, gerenciamento de perguntas e integração com Firebase.

**Tecnologias:** React, TypeScript, SCSS, Firebase Authentication e Firebase Realtime Database

[Repositório](https://github.com/CardosoRepository/Letmeask) | [Deploy](https://letmeask-db1a2.web.app)
